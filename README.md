# 二、改进效果
![image](https://github.com/DrJia/PicMgrDemo/blob/master/readme_src/picMgr.gif?raw=true)

1. 首先是可以上下左右拖动，解决了，只左右滚动，在选择后图片放大导致顶部被切割的问题。

3. 加入了仿微信拖动到某处删除图片的功能
